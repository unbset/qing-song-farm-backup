# 01_Knowledge — 长期知识记忆

> 长期知识记忆。支撑经验积累与 SOP 迭代。

## 用途

保存经过沉淀的经营知识、标准操作流程、最佳实践与专业经验。

## 内容范畴

- SOP（标准操作流程）
- 最佳实践
- 养殖/种植经验
- 设备知识与操作手册
- 供应商档案与评估
- 流程规范
- 经过复盘验证的经验总结

## 版本管理

支持版本迭代。文件命名时标注版本号：

```
SOP_育肥流程_v1.md
SOP_育肥流程_v2.md
供应商_饲料A_v1.md
```

旧版本保留以供追溯，用版本号区分。

## 与其他目录的关联

| 来源 | 关系 |
|------|------|
| 03_Incidents | 复盘总结经验 → 沉淀为 SOP/知识 |
| 02_Operations | 运营数据长期规律 → 提炼为最佳实践 |
| 00_Strategy | 战略目标 → 指导知识方向 |

## 生命周期

**长期保留。** 仅当知识被明确取代（如 SOP_v3 发布）时，可在归档后移除旧版。
