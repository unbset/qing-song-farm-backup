## 露营住宿阿米巴 — 产品表A筛查


### 产品1：散客露营

| 序号  | 体检项目  | 体检结果                                                                                                |
| :-- | :---- | :-------------------------------------------------------------------------------------------------- |
| 1   | 用户定义  | 潍坊市区20-40岁年轻群体（情侣、朋友聚会、亲子家庭），周末想找个有特色、能放松的户外空间，不想自己买装备                                              |
| 2   | 需求定义  | **痛点**：潍坊周边露营地同质化严重（草坪+帐篷，千篇一律）。**痒点**：想体验真正的“松林露营”——有树荫、有松针地毯、有自然气息。**嗨点**：在松林下躺着听风、孩子在地上捡松果、夜晚看星空 |
| 3   | 解决方案  | 按次收费的松林露营位租赁（含基础装备：帐篷/天幕/桌椅），客户可自带食品或向餐饮阿米巴采购餐食。松林下设置公共休息区和拍照打卡点                                    |
| 4   | 竞争力定义 | **差异化在“松林”本身**。潍坊一千多棵松树林是独一无二的资源，不是人造草坪能复制的。松林提供的是天然遮阴、松针的触感和气味、林间的光影——这些是花钱建不出来的                   |
| 5   | 支持系统  | 松林资源（已有）、基础露营装备（需采购）、露营区维护人员。核心能力：场景维护和氛围营造                                                         |
| 6   | 排兵布阵  | 战场：短视频（松林光影+客户体验画面）→ 线上预订/到店 → 现场接待。武器：松林本身的视觉冲击力就是最好的素材                                            |
| 7   | 战果复盘  | ✅ 2024年已有露营经营基础。样板测试已通过                                                                             |
| 8   | 飞轮定位  | 主定位：**引流入口 + 利润中心**。松林露营是农场最具视觉辨识度的场景，天然适合短视频传播引流。按次收费产生直接现金流                                       |
| 9   | 系统贡献  | 为农场提供最具辨识度的场景标签（“潍坊松林露营地”）；为餐饮、自然体验输送客流；为长搭露营转化种子客户。不可替代（松林是农场的核心资源）                                |
| 10  | 底线承受  | 最坏情况：客户在露营地受伤（跌落/火灾/恶劣天气）。承受能力：单次事故可承受（保险+赔偿），但重大安全事故不能承受。预防：安全须知+消防设施+恶劣天气关闭露营区+购买场地责任险            |

**表A结论**：✅ 全部通过。进入表B。


### 产品2：长搭露营

| 序号 | 体检项目 | 体检结果 |
| :--- | :--- | :--- |
| 1 | 用户定义 | 潍坊本地有一定消费能力和闲暇时间的人群：自由职业者、退休银发族、有幼儿的全职妈妈。已有十几户付费验证，年费3600元 |
| 2 | 需求定义 | **痛点**：城市里没有一个“属于自己的户外院子”。**痒点**：想要一个可以随时去的自然空间，按自己喜好布置。**嗨点**：“我在松林里有个小院”——这种归属感和身份标签 |
| 3 | 解决方案 | 在松林下划分独立小院（围栏+水电+基础平台），年租给客户。客户可自行搭建帐篷/天幕、摆放家具、储存个人物品。农场负责公共区域维护 |
| 4 | 竞争力定义 | **不是租一块地，是租一种“松林下的生活”** 。城市周边没有任何产品能提供一个“属于你的松林小院”。这个供给在潍坊是唯一的 |
| 5 | 支持系统 | 松林资源、已有十几户验证、水电配套经验。核心能力：小院维护+长搭社群运营 |
| 6 | 排兵布阵 | 战场：现有长搭客户口碑传播 + 短视频展示“我的松林小院”真实生活。武器：长搭客户自发的朋友圈和短视频（每一个小院都是一个活广告） |
| 7 | 战果复盘 | ✅ **已通过样板测试**。十几户付费客户，年费3600元/年，产品市场验证已完成 |
| 8 | 飞轮定位 | 主定位：**复购锚点 + 传播杠杆**。长搭客户是最稳定的高频到访群体，每次到访都可能连带餐饮、农产品消费。且每个长搭小院都是农场的“活广告” |
| 9 | 系统贡献 | ①提供稳定的年费收入；②创造“人气基底”——任何时候农场都有长搭客户在，不会冷清；③长搭客户的社交圈是高质量的口碑传播网络；④长搭客户的小院可发展为“民宿”的房源池。不可替代 |
| 10 | 底线承受 | 最坏情况：长搭客户之间纠纷（噪音/边界/物品丢失）；小院设施损坏责任争议；客户因个人原因退租。承受能力：均可承受。预防：租赁合同明确权责+小院使用公约+退租条款（提前1个月通知，按剩余月份退款） |

**表A结论**：✅ 全部通过。进入表B。


### 产品3：住宿（过夜）

| 序号 | 体检项目 | 体检结果 |
| :--- | :--- | :--- |
| 1 | 用户定义 | 远道而来的客户（潍坊周边县市/外地游客）、深度体验客户（参加假期农事营的家庭）、不想赶夜路的团建企业 |
| 2 | 需求定义 | **痛点**：玩了一天不想赶回去，但附近没有有特色的住宿。**痒点**：想体验“在松林里睡一觉”的感觉。**嗨点**：夜晚的松林、星空、清晨的鸟叫 |
| 3 | 解决方案 | 分两条腿走：①自营：农场提供全配套露营装备（帐篷+睡袋+洗漱设施），客户拎包入住；②众筹/共建：引入外部资金或客户共建小木屋/土团屋，建成后作为住宿运营，共建者享有分红或使用权。同时，长搭客户可将自己小院出租给过夜客人，农场提供撮合和分成 |
| 4 | 竞争力定义 | **松林过夜的体验在潍坊是独一份**。但当前阶段无差异化优势——潍坊周边已有专业露营地可过夜。未来的差异化在于土团屋/小木屋等特色住宿载体 |
| 5 | 支持系统 | ⚠️ **支持系统不完善**。自营住宿需要布草更换、夜间值班、洗漱设施、安全管理。当前团队不具备这些能力。众筹/共建模式尚在设想阶段，无验证 |
| 6 | 排兵布阵 | 战场：等特色住宿载体建成后再推广。武器：土团屋/小木屋本身就是传播素材 |
| 7 | 战果复盘 | ⚠️ **未做样板测试**。长搭客户小院出租模式可先试跑（已有房源，零成本启动） |
| 8 | 飞轮定位 | 主定位：**传播杠杆 + 利润中心（远期）**。过夜客户停留时间更长，消费场景更多，传播产出更丰富（星空、清晨、篝火） |
| 9 | 系统贡献 | 延长客户停留时间→增加消费机会→提升客单价。但当前阶段运营成本高，系统贡献为负 |
| 10 | 底线承受 | 最坏情况：住客安全事故（火灾/煤气/跌倒）；客户财物丢失；邻里投诉（噪音）。承受能力：单次事故可承受，重大安全事故不能承受。预防：消防设施+安全须知+夜间值班+保险 |

**表A结论**：✅ 通过，但标记为**低优先级辅助产品**。建议分阶段：①先跑通长搭客户小院出租分成模式（零成本验证）；②众筹共建土团屋（风险共担）；③最后考虑自营住宿（投入最大）。


### 产品4：团建/宴会活动组织

| 序号 | 体检项目 | 体检结果 |
| :--- | :--- | :--- |
| 1 | 用户定义 | 潍坊本地中小企业（20-50人）、银行/保险分支、政府单位、学校班级。决策者：行政/HR/工会负责人 |
| 2 | 需求定义 | **痛点**：每年团建就是吃饭唱歌，员工腻了，HR没新意。**痒点**：想要一个“有特色、有意义、能发朋友圈”的团建活动。**嗨点**：团队一起在松林下完成挑战、自己动手做窑鸡、晚上篝火晚会 |
| 3 | 解决方案 | 标准化团建套餐：半天版（团建活动+特色午餐）、全天版（上午活动+特色午餐+下午活动）、过夜版（全天版+松林过夜+篝火晚会）。不接定制需求，只卖套餐 |
| 4 | 竞争力定义 | **差异化在“松林场景+生态美食+自然体验”的打包组合**。市区团建公司只能做活动没场景，普通农家乐只有吃饭没活动。我们有场景+活动+美食的完整链条 |

| 7 | 战果复盘 | ✅ 2024年已有团建接待基础。样板测试已通过 |
| 8 | 飞轮定位 | 主定位：**利润中心**。高客单价（人均150-300元），工作日填充（企业团建多在周中），提高场地利用率 |
| 9 | 系统贡献 | ①工作日现金流（弥补周末-工作日落差）；②一次性触达20-50人，其中部分转化为散客/长搭/认养客户；③企业客户是礼盒产品的优质渠道 |
| 10 | 底线承受 | 最坏情况：活动组织失误（安全事件/客户不满意拒付款）。承受能力：单次可承受，赔偿或退款。预防：标准化流程降低失误率+安全须知+保险+签约时明确取消和赔付条款 |

**表A结论**：✅ 全部通过。进入表B。

## 商业评估：该阿米巴现有产品是否足以保障运营？

**结论：产品结构合理，但缺少一个“轻量转化钩子”。** 散客露营引流，长搭露营锁客，团建冲量，住宿远期布局——结构完整。但散客露营客户离场后，缺少一个“带点什么走”的产品。

**补充建议：新增一个辅助产品——“露营烧烤食材包”。**

| 项目 | 内容 |
| :--- | :--- |
| 产品名称 | 露营烧烤食材包 |
| 产品定位 | **利润款 + 传播杠杆** |
| 目标客群 | 散客露营客户（核心）、长搭客户（日常采购） |
| 产品内容 | 向餐饮阿米巴采购的半成品烧烤食材组合：腌制好的散养鸡翅/鸡腿、现摘蔬菜串、自磨面粉小面包、调味料包。客户下单后餐饮厨房备货，露营区领取 |
| 定价 | 双人包88-128元，家庭包（4人）168-228元 |
| 为什么建议做 | ①**提升客单价**：散客露营场地费有限，食材包能大幅拉升单客消费；②**解决客户痛点**：客户不用自己采购+腌制+串串，降低露营决策门槛；③**内部协作**：露营阿米巴向餐饮阿米巴采购，两个阿米巴都有收入，协作有动力；④**传播素材**：松林下烧烤的画面是优质短视频素材 |
| 与现有产品的关系 | 向餐饮阿米巴采购食材（内部转移价），露营阿米巴加价销售。是露营体验的升级选项，不单独售卖 |
| 启动时机 | 与散客露营同步推出 |

## 露营住宿阿米巴 — 补充运营建议

**1. 与餐饮阿米巴的内部采购规则必须书面化。**
露营住宿阿米巴的三个产品（团建餐、烧烤食材包、露营客人散点餐）都涉及向餐饮阿米巴采购。建议建立“周度预报+当日追加”机制：露营阿米巴每周五向餐饮阿米巴预报下周团建和露营预订量，餐饮据此备货；当日散客追加订单，餐饮在能力范围内满足。内部转移价以餐饮阿米巴的成本价上浮10%-15%结算，确保两个阿米巴都有利润空间。

**2. 长搭客户是农场的“压舱石”。**
长搭客户月均到访3次以上，每次可能产生餐饮、农产品消费。30户长搭客户每月仅连带消费就可能贡献数万元。经营长搭客户的核心不是收租，是社群运营——让他们觉得“这个小院是我的第二个家，这个农场是我的圈子”。建议：每月一次长搭专属活动（成本极低但体验感强），每季度一次长搭客户意见征集（让他们参与农场决策，强化归属感）。

**3. 露营烧烤食材包是内部协作的试金石。**
这个产品做好了，露营住宿和餐饮两个阿米巴都有利润，协作有动力。做不好，就是互相抱怨（露营嫌餐饮备货慢，餐饮嫌露营订单零散）。建议先做1个月试跑，每周复盘，跑通后再正式纳入产品线。
