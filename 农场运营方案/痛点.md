# 近郊游
- 吸引客人频繁回购，形成稳定的生活仪式感。--->为客户提供什么价值？他们愿意来。
- 近郊游消费，很多钱花在停留体验上。
- 过去人们追求远方，现在人们追求恢复（放松、自我修复、情绪价值、生活方式）。



# 农场的经营难点
- 找到一群能够安安心心的、愿意付出的工作人员。不是那种拿工资的。
- 非常依赖：情绪表达、审美、个人感受、真实体验。不是特别依赖资源。



# 农场的价值
- 人在自然里，更容易听见自己的真实状态。
- 山野自然性适合：徒步、安静、重新整理思绪。
- 小镇疗愈型适合：慢生活、咖啡馆、文艺感。适合写东西、拍图文、录口播、做生活感内容。
- 不是卖景点，卖的是氛围感、松弛感、逃离感、治愈感
- 年轻人的目的：拍照、放空、恢复情绪、内容创作、聚会
- 近郊游最缺的是真正有内在表达的人。大量账号都在拍“这里好看”，很少有人讲“为什么人需要离开城市、为什么成年人越来越疲惫、为什么自然能修复情绪、为什么一个人的旅行很重要”
- AI时代，人会更渴望身体感、真实连接、自然、安静、面对面体验。未来的徒步、营地、小聚、线下疗愈、慢旅行会更火。


# 农场的赢利点
- 内容种草（门槛最低）：周边推荐、一日游、治愈vlog、一日路线、咖啡+自然。【广告、团购、合作】
- 民宿/营地（重资产、利润高）：需要运营、审美、流量。【森林小屋】
- 小团路线：区别传统旅行团，小众、有氛围感、有主题、有同频感【周末疗愈徒步、城市逃离计划、创作者山野营】
- 个人品牌（长期价值）：卖的不是旅行，而是“跟这个人一起生活的感觉”。比如一个会寻找安静地方的人、一个会观察生活的人、一个有思想的人、一个能表达情绪的人。形成强的长期信任。


AI时代与农场：
- AI越强，人的真实体验反而越值钱。AI越强，人越渴望线下真实。
- AI时代，近郊游应属于情绪行业。在AI行业，行程规划、酒店筛选、攻略、预算优化都由AI完成，真正值钱的会变成
	- 情绪价值（放松感、被理解、陪伴感、治愈感、共鸣等）。
	- 人的真实体验
	- 个体人格：未来大家不是跟攻略走。而是跟人走。比如：我喜欢这个人的生活节奏、我认同他的观察方式、我喜欢他寻找安静地方的方法。   人格=新时代旅行品牌。
- AI时代，旅行内容会在AI的自动生成辅助下，千篇一律的模块化内容会严重泛滥，比如这里超美、周末一定要来等等，因为AI可以无限生成。未来值钱的是：
	- 真实感：人的真体验，不是AI的拼凑。  包含：不完美、有情绪、有真实观察、有人生经历。
	- 深度表达：未来大家不缺景色，缺的是“有人帮我理解生活”。“旅行+思考”是方向。
	- 小而真实的人格IP
- 过去卖景点，现在卖生活方式，未来卖精神状态。  在AI时代，信息更密集、工作更卷、注意力更碎片化，人的精神疲劳会更严重，所以“恢复类体验”会成为巨大需求。
