# 04_Agent_Runtime — Agent 运行记忆

> 短期运行记忆。建议保留 7-30 天。

## 用途

记录 AI Agent 的执行日志、工作流状态、调度记录与运行情况，用于 Agent 自我优化。

## 内容范畴

- Agent 执行日志
- 工作流状态记录
- 调度记录
- Agent 失败案例
- 自动总结报告
- Memory 调用日志
- 决策推理过程（关键决策可追溯）

## 生命周期

| 类型 | 保留期 | 说明 |
|------|--------|------|
| 普通执行日志 | 7 天 | 自动清理 |
| 失败案例分析 | 30 天 | 分析完成后可提前清理 |
| 关键决策推理 | 归档到 00_Strategy | 重要决策转入长期记忆 |
| Agent 优化建议 | 30 天 | 评估后执行或丢弃 |

## 与其他目录的关联

| 输出方向 | 说明 |
|----------|------|
| 01_Knowledge | Agent 发现的规律可沉淀为知识 |
| 00_Strategy | Agent 的战略建议可归档 |
| 03_Incidents | Agent 发现的异常触发 Incident |

## 文件命名规范

```
YYYY-MM-DD_Agent名_操作类型.md
```
