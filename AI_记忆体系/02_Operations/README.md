# 02_Operations — 短中期运营记忆

> 短中期运营记忆。建议保留 30-90 天。

## 用途

记录日常运营数据、采购、销售、环境监测、任务执行等操作层面的信息。

## 内容范畴

- 日常采购记录
- 库存变动
- 销售记录
- 环境数据（温度、湿度、通风等）
- 任务分配与执行记录
- 运营日志
- 日常巡检记录

## 数据流转

运营数据是知识沉淀的原料。运营记录经过分析后：

```
运营数据 → 发现规律 → 提炼最佳实践 → 沉淀到 01_Knowledge
运营异常 → 触发 Incident → 复盘 → 经验 → 01_Knowledge
```

## 生命周期

| 类型      | 保留期  | 清理后        |
| ------- | ---- | ---------- |
| 日常日志    | 30 天 | 自动清理       |
| 采购/销售记录 | 90 天 | 汇总到月度报表后清理 |
| 环境数据    | 90 天 | 提取趋势后清理    |
| 任务记录    | 30 天 | 完成归档后清理    |

## 文件命名规范

```
YYYY-MM-DD_运营日志.md
YYYY-MM_采购汇总.md
YYYY-MM_销售记录.md
```
