
## 1.阿米巴清单
农场一共设6个阿米巴，分别是：
1. 生态种养：利用堆肥和发酵技术，把种植和养殖结合起来。主要意义在于增加农场的宣传亮点，并增加客户对农场食材的信任和向往。
	工作内容：种植、养殖、堆肥、酵素、种养循环理念的对外宣传、初级农产品的定价。

2. 露营住宿：我们农场最大的特色是一千多棵松树林，在潍坊绝无仅有。利用松林资源，打造林下露营与林间小屋。
	工作内容：露营区的维护、场景打造；露营及住宿的宣发、客户维护；团建、宴会的接待及服务；带着户外的客人做游戏、烘托气氛等。


3. 餐饮：保障在农场就餐人员的餐食。
	工作内容：厨房的维护；室内餐的运营；菜品研究；初级农产品的二次加工。

4. 自然体验：纯轻资产，可使用农场中的一切资源，设计为体验课程。
	工作内容：结合农场的资源，研发体验课程；自然体验课程的客户接待及维护；自然体验课程的市场拓展。

5. 田园摄影：借助农场已打造的场景，开展田园摄影业务。比如后期农场有了一些列不同风格的小木屋、土团屋等，有农作、花海等场景，可以顺便开展田园摄影。可以老年团、儿童摄影等为主。

6. 农场运营：查漏补缺，其他5个阿米巴工作范围之外的事情，交给农场运营阿米巴。比如外采、场地租赁、品牌运营等。
	工作内容：相当于农场办公室，统一调度，什么都干。主要负责农场各种产品的统一对外销售；会员系统维护；资质维护；拟定经营目标，下发考核与监督等等。

备注：每个阿米巴负责各自板块的新媒体运营，负责各自板块相关产品的sop制定。

## 2.阿米巴产品清单
### 1.生态种养产品清单：
- 初级农产品销售：对内销售给内部阿米巴；对外销售给客户。
- 菜园租赁：其实潍坊的菜园租赁挺多的，价格也比较低，但是也不能没有。不建议耗费很多精力，可以作为一个附加产品（比如租给长搭用户），不作为捆绑用户粘性的产品使用。
- 家禽家畜认养：我感觉这个产品比菜园租赁的市场要好一些。
- 酵素与堆肥零售：后期堆肥和酵素技术成熟后，可以做成产品卖给客人。


### 2.露营住宿产品清单：
- 散客露营：散客主要是指来农场游玩、体验露营的客人，主要以场地租赁为主，按次收费。
- 长搭露营：长搭是指按月或按年租赁松下小院的客人，由农场在松树下围一方小院，配套水电租给客户。目前农场已有十几户，年费是3600元/年。
- 住宿：住宿有两种，一种是农场配套全套的露营装备，客人来农场过夜。另一种是众筹模式，比如长搭客户通过农场把自己的小院租给过夜的客人，农场给长搭客服支付分成。同理，我想以共建的模式，众筹一部分农场小木屋或土团屋，作为住宿的运营设施。
- 团建/宴会活动的组织：这不用解释了吧。
### 3.餐饮产品清单
- 室内餐：农场的室内餐环境比较好，当初装修的很高档，桌椅配套齐全，作为会所使用。室内由餐饮阿米巴经营，室外由露营阿米巴经营。
- 农产品的二次加工：比如减脂餐等。
### 4.自然体验产品清单
- 各种体验活动：比如农作物的种植、收获与加工；日用品的手作；工艺品的手作等等。
- 手作产品的销售：比如手工香、日用品等。
### 5.田园摄影产品清单
- 摄影：给人照相，兼职摄影师，按次收费。
### 6.农场运营产品清单
- 农产品品牌运营

### 1.生态种养阿米巴 — 产品快速排序

|产品|排序|理由|
|---|---|---|
|**初级农产品销售**|核心|阿米巴的立身之本。对内供应餐饮和农场运营，对外产生现金流。内外两条线都要跑通|
|**家禽家畜认养**|核心|你判断市场比菜园租赁好，且认养自带复购钩子（定期收蛋/探访），是天然的锚点款|
|**酵素与堆肥零售**|辅助|技术成熟后可做，但市场需要教育，短期不是主力收入。酵素营养液5元/瓶作为传播杠杆效果更好|
|**菜园租赁**|辅助|你已主动降级，不投入过多精力。作为长搭用户的附加权益是合理定位，不单独推广|

**排序说明**：生态种养的主定位是信任引擎，但作为独立阿米巴必须有收入来源。初级农产品销售是收入基本盘，家禽认养是利润+复购锚点，这两个必须做成。酵素零售是传播杠杆，低成本维持即可。菜园租赁按你的判断不追加投入。

### 2. 露营住宿阿米巴

| --------- | ------ | --------------------------------------------------------------------- |

| 长搭露营      | **核心** | 已有十几户付费验证，年费3600元。这是天然的复购锚点——长搭客户反复到访，连带餐饮、农产品消费。且长搭客户的小院本身就是农场的“活广告” |
| 团建/宴会活动组织 | **核心** | 高客单价，工作日填充，提高场地利用率。与餐饮阿米巴协作（互相采购），是利润中心的重要组成部分                        |

|           |        |                                                                       |

### 3. 餐饮阿米巴

| 产品      | 排序     | 理由                                                            |
| ------- | ------ | ------------------------------------------------------------- |
| 室内餐     | **核心** | 现有高档装修是沉没成本，必须用起来。室内餐环境好，定位会所式服务，客单价高，是利润中心的核心                |
| 农产品二次加工 | **核心** | 你提出的减脂餐方向，逻辑成立——把初级农产品转化为高客单价解决方案，是该阿米巴利润增长的关键。但需先验证市场需求和物流成本 |

**说明**：餐饮阿米巴产品线少但都是核心。室内餐保基本盘，二次加工拓利润空间。窑烤宴、蒸笼宴等特色餐食的具体形式，建议在深度分析时作为“室内餐”或“室外餐”（与露营阿米巴协作）的子产品展开。

### 4. 自然体验阿米巴

| 产品     | 排序     | 理由                                                                                   |
| ------ | ------ | ------------------------------------------------------------------------------------ |
| 各种体验活动 | **核心** | 纯轻资产，利用农场现有资源。引流（低价体验课）+ 信任深化（深度课程）+ 利润（假期农事营），一个产品线可拆出多层价值。且体验活动是农场“修复人与自然关系”最直接的载体 |
| 手作产品销售 | **核心** | 手工香、日用品等，原料来自农场，故事性强，传播杠杆价值高。客户带回家就是广告。且手作产品可同时走两条路：体验课现场销售 + 农场运营阿米巴统一对外销售          |

**说明**：自然体验阿米巴的两个产品都是核心。体验活动是引流+信任引擎，手作产品是传播杠杆+利润。两者相辅相成——体验课产出产品，产品带动体验课报名。

---

### 5. 田园摄影阿米巴

|产品|排序|理由|
|---|---|---|
|摄影|**待定**|你定位“顺便开展”，依赖场景积累（小木屋、土团屋、花海等）。当前农场场景还在建设中，摄影业务缺乏差异化竞争力。建议等至少3-5个成熟拍摄场景落地后再启动，现阶段不投入精力|

**说明**：田园摄影作为传播杠杆的价值是明确的，但启动时机取决于场景成熟度。标记为“待定”而非“建议砍”，是因为它与农场长期方向一致，只是现在不该做。

---

### 6. 农场运营阿米巴

| 产品      | 排序     | 理由                                                                                                |
| ------- | ------ | ------------------------------------------------------------------------------------------------- |
| 农产品品牌运营 | **核心** | 这是农场运营阿米巴存在的核心意义——统一对外销售接口。向生态种养采购农产品、向自然体验采购手作产品、向餐饮采购减脂餐，以统一品牌形象对外输出。没有这个阿米巴，品牌碎片化、销售效率低、客户数据分散 |

**说明**：农场运营阿米巴只有一个产品，但这个产品是整合所有阿米巴产出的中枢。它不生产，只采购+销售+维护客户关系。会员体系、资质维护、目标考核等是其内部职能，不单独作为产品列出。
