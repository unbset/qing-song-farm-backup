# 00_Strategy — 长期战略记忆

> 永久长期记忆。不得自动删除。

## 用途

保存农场的长期经营战略、定位、发展规划与核心原则。

## 内容范畴

- 长期经营目标
- 农场定位与愿景
- 自动化规划路线
- 风险偏好与风控策略
- 经营原则与价值观
- AI 发展路线图
- 战略决策记录（含决策依据）

## 生命周期

**永久保留。** 本目录的信息支撑农场未来 3-5 年甚至更长周期的经营方向，任何自动化清理不得触及此目录。

## 与其他目录的关联

| 来源 | 关系 |
|------|------|
| 03_Incidents | 系统性风险事件可能触发战略调整 |
| 01_Knowledge | 长期积累的经验可能反哺战略优化 |
| 04_Agent_Runtime | Agent 的战略推演记录归档于此 |

## 文件命名规范

```
YYYY-MM-DD_主题.md           — 单次战略决策/规划
System_Principles.md          — 系统核心原则（固定文件）
Roadmap.md                    — 路线图（持续更新）
Risk_Policy.md                — 风险政策（持续更新）
```
