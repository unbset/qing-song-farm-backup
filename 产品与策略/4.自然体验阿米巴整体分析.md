# 自然体验阿米巴 — 整体分析

> **分析日期**：2026-05-27
> **分析框架**：青松农场业务板块分析框架 V1.0
> **产品清单**：各种体验活动、手作产品销售


## 第一部分：业务定位分析

| 分析项 | 内容 |
| :--- | :--- |
| **阿米巴名称** | 自然体验 |
| **在飞轮中的位置** | 引流入口 + 信任引擎 + 传播杠杆 |
| **主定位** | **引流型**。这是农场降低首次到访门槛的最佳工具——“带孩子来做一瓶松针纯露”比“来农场逛逛”有吸引力得多。体验课是农场的“试用装”，让潜在客户以低门槛体验农场核心价值，进而转化为餐饮、露营、认养等其他产品的客户 |
| **辅助作用** | 信任引擎（深度课让客户亲身体验酵素制作、土壤观察、农事劳动，信任建立远超被动观看）；传播杠杆（孩子举着作品的笑脸是最强传播素材）；利润中心（深度课和假期营产生直接利润） |
| **业务等级** | **A级—核心业务**。缺失则农场失去最有效的引流工具，获客成本将大幅上升。且农场“教育”维度的价值缺失，“修复人与自然的关系”这一使命缺少了最直接的传递载体 |
| **承接谁** | 短视频/内容获客（课程精彩片段引流）；餐饮阿米巴和露营住宿阿米巴（到店客户转化为课程学员）；农场运营阿米巴（线上售卖体验课） |
| **输送谁** | 向餐饮阿米巴输送用餐客流（学员和家长午餐需求）；向生态种养阿米巴输送农产品购买需求（课后顺手买菜）；向露营住宿阿米巴输送团建活动内容和散客转化；向农场运营阿米巴输送手作产品货源和会员转化 |
| **如果缺失会怎样** | ①获客手段只剩“来农场逛逛”，吸引力不足，短视频内容单一；②客户信任建立路径变长（只能看不能做）；③孩子群体吸引力下降，亲子家庭这一核心客群流失；④手作产品的故事性和体验性大打折扣 |
| **替代方案** | 可部分替代。若无法自建自然体验团队，可引入外部自然教育机构合作（对方出课程和老师，农场提供场地和资源，收入分成）。代价是品控和客户体验不完全可控，且利润空间被压缩 |
| **现在该不该做** | **初创期（活下来）**。这是第三梯队中需要尽快启动的阿米巴——它不直接产生最大现金流，但它为餐饮和露营住宿输送客流，为整个飞轮提供“人的入口”。建议在餐饮和露营住宿SOP稳定后，立即启动引流课样板测试 |


## 第二部分：产品与客户分析

| 分析项 | 内容 |
| :--- | :--- |
| **目标客群画像** | 亲子家庭（3-12岁孩子，决策者是妈妈——周末需要“有意义、能发朋友圈、不太累”的遛娃方案）；幼儿园/小学班级游学团（决策者是老师/家委会，需要教育意义+安全+价格合理）；银发群体（手作体验，有时间有情怀）；团建企业（向露营阿米巴输出活动内容） |
| **客户真正买的是什么** | **“孩子的成长体验+家长的精神喘息”** 。家长买的不是一节课，是“孩子在大自然里专注做一件事的样子”和“我可以坐在旁边喝杯茶不用管他的片刻”。孩子买的是“我做了一瓶松针纯露带回家”的成就感和自豪感 |
| **客户决策逻辑** | “周末带孩子去哪→刷到体验课视频→价格不贵/孩子能学到东西/有作品可带走→比去商场有意义→报名”。第一推动力是“孩子的体验感+家长的省心”，替代方案是商场手工坊/游乐场/公园/在家看iPad |
| **产品层级设计** | **引流款**：各种体验活动-引流课（68元/人，1-1.5小时）。**核心款**：各种体验活动-深度课（298元/人，全天含午餐）。**利润款**：各种体验活动-假期营（1980-2980元/人，7-10天含住宿餐食）。**传播款**：手作产品销售（28-58元/件，品牌可带走载体） |
| **定价逻辑** | 引流课：低成本体验价，覆盖材料成本+少量利润，目标是降低决策门槛而非赚钱。深度课：价值定价，对标潍坊本地一日游学产品（200-400元区间），溢价来自“真农场真生态”的差异化体验。假期营：市场对标+价值定价，对标同类自然教育营（2000-4000元区间），中位定价确保竞争力。手作产品：市场对标手工市集/精品店同类产品，溢价来自“可溯源的农场原料” |
| **与农场核心价值的关系** | **这是“修复人与自然的关系”最直接的载体**。孩子亲手触摸土壤、制作酵素、采摘作物、喂养动物——这些体验是城市生活无法提供的。自然体验阿米巴将农场“修复人与自然关系”的抽象使命，转化为可参与的、可感知的、可带走的具体体验 |
| **是否可标准化** | **65%**。课程教案、流程、话术、材料清单可标准化。课程中与孩子的互动和气氛调动需一定教育经验和亲和力，这部分难以完全SOP化 |
| **用户为什么复购** | 三级递进机制：引流课（低门槛尝试）→深度课（更深体验）→假期营（沉浸式），每一级都是上一级的自然延伸。家长群内每周的自然观察日记和课程预告保持连接。深度课老学员报假期营享折扣 |
| **有没有自传播能力** | **极强**。孩子举着作品的笑脸、专注做事的侧脸、和小动物的互动——这些都是家长必发朋友圈的内容。老师课后发送的抓拍照片是家长的“社交货币”。结营仪式的仪式感画面传播力极强 |


## 第三部分：运营与标准化分析

| 分析项 | 内容 |
| :--- | :--- |
| **端到端流程** | 以引流课为例：线上报名→提前1天确认人数→准备材料→客户到店签到→老师引导至教学点→开场破冰→主体活动（引导+辅助+抓拍）→作品展示→结课引导（推荐深度课/购买手作产品/加微信拉群）→课后发送照片+跟进 |
| **运营强度** | **中高**。课程执行中老师需全程高度专注（控场+教学+安全+抓拍）。体验课集中在周末，工作日可用于课程研发、材料准备、客户跟进。假期营期间运营强度极高（全天候管理） |
| **关键触点** | ①开场破冰时老师的亲和力（决定孩子是否投入）；②主体活动中孩子的参与度和成就感（决定家长满意度）；③作品展示环节的仪式感（决定传播素材质量）；④课后发送的照片和跟进（决定复购） |
| **人员需求** | 自然体验老师1人（核心岗位，负责课程研发和执行），助手1人（周末按需，协助物料准备和现场拍照）。假期营期间需增加1-2名临时助教（大学生志愿者或兼职老师） |
| **标准化程度评估** | **65%**。可标准化的：课程教案（逐步骤+逐话术）、材料清单、安全预案、拍照要点、课后跟进SOP。依赖经验的：课程中与孩子的互动、临时状况应变、气氛调动 |
| **关键岗位依赖度** | **中高**。自然体验老师兼具课程研发和教学执行，离职影响课程质量和排课。应对：①所有课程教案书面化至“逐句话术”级别，降低研发依赖；②培养助手逐步上手基础教学（先从引流课开始带）；③与潍坊本地自然教育从业者、幼儿园老师、牧校毕业生建立联系，建立兼职/后备老师库；④言渊本人可作为储备讲师（他的经典解读和人生经历是深度课的最佳素材） |
| **培训成本** | 有教育或农业背景的新人：3-4周跟岗可独立带引流课。无相关背景新人：6-8周。培训材料（教案手册+教学视频+安全手册）需在Q2-Q3建立 |
| **SOP状态** | ⚠️ **未建立**。2024年有零散课程执行但没有标准化教案。Q2-Q3必须完成：引流课标准教案2套、课程执行SOP、安全预案、课后跟进SOP |

### 4.5 财务分析结论

1. **轻资产高回报。** 初始投入仅约1.5万，年结余约25万，投资回报率极高。核心原因是利用农场现有资源（场地、素材、原料），几乎零固定资产投入。
2. **引流课是收入的压舱石。** 年贡献约20万收入，占总收入近一半。但它也是成本中心——引流课毛利率约65%，低于深度课和假期营（50%-55%）。引流课的价值不在自身利润，在于为其他产品输送客户。
3. **假期营是利润的加速器。** 3期假期营年收入8万，仅需2-3个月集中交付，边际成本低，是暑期现金流的重要补充。
4. **手作产品是长尾利润。** 年收入4万，毛利率70%以上，且不受季节影响（线上可全年销售）。规模虽小但利润率高，是品牌传播的有效载体。
5. **整体盈利能力良好。** 毛利率约59%，年结余约25万，在农场各阿米巴中排名第三（低于露营住宿和餐饮，但远高于生态种养）。


## 第五部分：风险评估

| 分析项 | 内容 |
| :--- | :--- |
| **风险清单** | ①课程中孩子受伤（工具使用/户外跌倒/蚊虫叮咬过敏/中暑）；②自然体验老师离职；③课程报名不足（引流课报不满，深度课凑不齐）；④天气影响户外课程；⑤手作产品质量问题（发霉/变质/过敏）；⑥假期营安全管理 |
| **最坏情况推演** | **场景一：课程中孩子受伤。** 孩子在使用工具时割伤/在户外奔跑时跌倒骨折→家长投诉/索赔→社交媒体负面传播→课程停开→品牌受损。承受能力判断：**不能承受**。预防：安全须知签收+保险全覆盖+危险环节替代方案+急救包+老师急救培训+每堂课安全员配置。**场景二：自然体验老师离职。** 唯一的核心老师离职→课程无法执行→排课中断→客户退费→口碑受损。承受能力判断：短期可承受（暂停排课2-4周），长期需有后备方案。预防：教案书面化+助手培养+后备老师库+言渊储备讲师 |
| **预防措施** | 安全：安全须知签收制度、课程设计排除高风险环节、急救包标配、老师急救证培训、每堂课安全员配置（助手或志愿者）、场地责任险+意外险。老师离职：教案逐句话术化、助手跟课培养、后备老师库建立、离职提前1个月通知。报名不足：课前48小时确认人数，未达最低人数（6人）则取消或合并场次，提前通知客户改期或退款 |
| **应急预案** | 孩子受伤：立即急救处理→通知家长→视情况送医→保险理赔→事后复盘调整安全预案。老师突然离职：言渊或后备老师顶上维持基础排课（优先保障引流课）→1个月内招聘到位→新老师跟课过渡。天气影响：户外课遇雨天转为室内备选方案（手作课替代农事课）→提前24小时通知客户→客户可选择改期或退款 |
| **退出标准** | 暂不设整体退出。自然体验是农场“教育”维度的核心载体，除非农场整体转型不做体验教育，否则不退出。单项退出：特定课程连续3个月报满率<50%则暂停该课程并替换；手作产品单品连续3个月月销<10件则下架 |

| **经验沉淀计划** | 每门课程执行满10次后做一次课程复盘（教案迭代+安全预案更新+客户反馈整合）。每季度更新一次课程目录（淘汰低分课程，新增测试课程）。所有教案和课程执行经验沉淀至hindsight |

## 自然体验阿米巴 — 核心结论

1. **轻资产高回报的引流引擎。** 初始投入仅约1.5万，年结余约25万。它是农场获客成本最低的渠道——“带孩子来做一瓶松针纯露”比任何广告都有吸引力。体验课客户转化为餐饮、露营、认养客户的自然转化率远高于纯线上获客。

2. **三级课程体系是客户价值深化的阶梯。** 引流课（降低门槛）→深度课（深化信任）→假期营（锁定长期关系），每一级都是上一级的自然延伸。这个体系让客户从“一次体验”逐步走向“长期关系”。

3. **手作产品是花小钱办长尾的品牌传播。** 年收入4万虽不高，但每件产品都是农场的可带走广告。客户用着松针纯露，朋友问“哪买的”——传播自然发生。且产品与体验课形成闭环（课上做→展示柜买→线上复购）。

4. **最大风险是老师离职和课程安全。** 自然体验老师是单人关键岗位，离职则课程中断。必须通过教案书面化至话术级别、助手培养、后备老师库建设三重机制对冲。课程安全是底线，安全预案和保险配置必须在开课之前就位。

5. **与露营住宿阿米巴的协同是关键增长点。** 自然体验向露营住宿的团建输出活动内容（内部结算），露营住宿的团建客户转化为自然体验的深度课和假期营客户。两个阿米巴在团建市场形成“场地+内容”的完整解决方案，这是任何单一竞品难以复制的组合。

6. **言渊是自然体验阿米巴的“王牌储备讲师”。** 他的经典解读能力、人生重建经历、儒释道融合智慧，是深度课和假期营的独特差异化资源。其他农场可以模仿松针纯露制作课，但无法模仿“言渊的围炉夜话”。这个资源应在课程体系中预留位置（如深度课的家长环节、假期营的夜谈环节），但不作为日常课程的依赖——言渊的时间精力有限，应集中在最高价值的场景使用。
