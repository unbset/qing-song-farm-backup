## 商业评估：该阿米巴现有产品是否足以保障运营？

**结论：基本足以维持现金流，但天花板明显。现有两个产品能活下来，但要活得好，需要一个补充产品。**

**现有产品结构分析**：

| 产品 | 角色 | 收入贡献 | 天花板 |
| :--- | :--- | :--- | :--- |
| 室内餐 | 利润中心（基本盘） | 高（周末翻台+团建宴席） | 翻台率受限于桌数和用餐时段；工作日闲置 |
| 农产品二次加工 | 利润中心（增量） | 待验证 | 依赖物流效率和客户积累，起量需要时间 |

**问题**：两个产品之间缺少一个“轻量级转化桥梁”。到店客户吃了饭，觉得好吃，想带点什么走——但礼盒在生态种养阿米巴，减脂餐需要提前预订配送。客户当下的冲动消费没有被承接。

**补充建议：新增一个辅助产品——“农场即食伴手礼”。**

| 项目 | 内容 |
| :--- | :--- |
| 产品名称 | 农场即食伴手礼 |
| 产品定位 | **利润款 + 传播杠杆** |
| 目标客群 | 到店用餐客户（吃完顺手买）、露营客户（带走当夜宵/次日早餐） |
| 产品内容 | 餐饮阿米巴利用厨房空闲时段制作的可即食/简单加热产品。例如：窑烤面包（农场自磨面粉+窑炉现烤）、风味辣酱（农场辣椒+生态种植的蒜）、腌菜/泡菜（农场时蔬制作）、卤鸡爪/卤蛋（散养鸡副产品） |
| 定价 | 窑烤面包15-25元/个，辣酱38-58元/瓶，卤味零食15-25元/袋 |
| 为什么是必须的 | ①**承接餐饮流量**：客户吃完饭的当下体验峰值最高，此时购买转化率最高。不承接，餐饮流量就只在餐厅止步；②**边际成本极低**：利用厨房空闲时段和农场自有食材，不需要额外设备或人力；③**传播杠杆**：客人带回家，朋友尝到问“哪买的”，传播自然发生；④**提升客单价**：一桌消费300元，走时带100元伴手礼，客单价提升33% |
| 与现有产品的关系 | 和室内餐共享厨房和食材，和减脂餐共享加工流程（分拣、包装环节SOP可复用） |
| 启动时机 | 室内餐SOP稳定后（Q2），先试2-3个单品，卖得好再加 |

## 餐饮阿米巴 — 补充运营建议

**1. 与露营住宿阿米巴的内部采购需明确**
窑烤宴的归属需定规则。建议：露营住宿阿米巴负责场地、氛围、服务，餐饮阿米巴负责出品。客人订窑烤宴，露营阿米巴向餐饮阿米巴以内部转移价采购餐食，再加价打包为“松林窑烤体验”整体产品。这样两个阿米巴都有利润，协作有动力。

**2. 减脂餐的物流是胜负手**
初期同城跑腿成本高（每单15-25元），吃掉了大部分毛利。必须尽快积累订单密度，从跑腿转向自有车辆集中配送。触发点：周订单≥30单时，自有配送成本可降至8-12元/单。建议初期用“满200免配送费”引导客户提高单次订购金额。

**3. 厨房人效最大化**
餐饮阿米巴的核心资源是厨房和厨师。室内餐的高峰在午晚市，厨房上午10:00-11:30、下午14:00-17:00相对空闲。这两个时段恰好可以用来做减脂餐分拣包装和即食伴手礼制作。同一套人马，三个产品的产出，人效最大化。
